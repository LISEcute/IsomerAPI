# IsomerAPI

Housing for the IsomerAPI project to be integrated with LISE++. This project is primarily self-contained and will work as an extension within LISE++.

IsomerAPI is based on two databases: isomer levels and gamma lines. It provides a GUI interface for working with isomer-level and gamma-line information.

The project runs in Qt Creator 16+ and compiles with Qt 6.7.3 using either MinGW or MSVC.
